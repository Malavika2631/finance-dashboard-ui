# Finlytics Dashboard

A multi-page finance dashboard UI built with pure HTML and CSS — no frameworks, no JavaScript. Focused on clean layout, visual hierarchy, and a realistic finance-app structure.

## Pages

| Page | Description |
|---|---|
| `index.html` | Landing/homepage |
| `dashboard.html` | Main dashboard view — account overview layout |
| `analytics.html` | Analytics/insights page |
| `profile.html` | User profile page |
| `transaction.html` | Transaction history / list view |

Each page has its own dedicated stylesheet (`homepage.css`, `dashboard.css`, `analytics.css`, `profile.css`, `transaction.css`) rather than one shared file, keeping each view's styling self-contained and easy to navigate.

## Tech Stack

HTML5 · CSS3 — no JavaScript, no frameworks or libraries.
